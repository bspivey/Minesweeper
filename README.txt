Description:

Contains main.cpp and main.exe for the Minesweeper Command Line v1.1 program.

Compile Instructions:

- Made for use in Windows 2000 or newer Windows OS due to commands for resizing the console window.
- Compile with C++11 due to use of to_string function.
- Use "-static" command for linking to ensure all necessary dll's are linked statically.
